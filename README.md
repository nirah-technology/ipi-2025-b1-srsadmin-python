# Bibliothèque de films

1. Il faut représenter un film (classe).
    - titre
    - date de sortie au cinéma
    - note 
    - résumé
    - genres
    - personnages

2. Il faut représente une biliothèque de films (classe).

3. Dans la bilitohèque, il faut permettre le trie des films par :
    - titre
    - date
    - note

4. Dans la bilitohèque, il faut permettre le filtrage des films par :
    - mot
    - note
    - date
